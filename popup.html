QuikScratch Guide

QuikScratch lets you type quick shortcuts to launch profiles, projects, studios and more.

    First, type scratch and press tab, then pick a shortcut
  • $ - Studio id - $1111
  • ! - Project id - !1111
  • @ - Username - @grahamsh
    • Or, type any url on scratch, for example create would bring you to https://scratch.mit.edu/create
